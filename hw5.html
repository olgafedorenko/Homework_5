Homework 5 - Array

Part 1 -Input

 A webpage containing a form that displays an input text box, radio buttons, checkboxes, and select items; along with a script that executes form validation.

Part 1b - Pulldown menu

 A webpage that contains a script that utilizes 2 pulldown menus. The first one should utilize an onchange event handler. The second should utilize a button with an onclick event handler to navigate to a page after an option is selected..

Part 2 - States

 Given the following table of data about several of our states from the 2010 census, create a webpage that enables the user to enter a state name (full or abbreviated) and have the State Name, State Capital, and population appear in separate form text fields of the page.

Part 3 - Salaries

 Use a one-dimensional array to solve the following problem: A company pays its salespeople on a commission basis. The salespeople receive $200.00 per week plus 9% of their gross sales for that week. For example, a sales person who grosses $5000.00 in sales in a week receives $200.00 plus 9% of $5000.00, or a total of $650.00. Create a webpage with a script (using an array of counters) that obtains the gross sales for each employee through a form and determines how many of the salespeople earned salaries

Part 4 - Numbers

 Create a webpage that contains a script, using a one-dimensional array to solve the following problem: Read in 10 numbers, each of which is between 10 and 100. I suggest using and re-using one input text box and displaying a count of the number of numbers entered. Another alternative would be having 10 input text boxes. As each number is read, print it only if it is not a duplicate number that has already been read.

Extra credit - Airlines Reservation System

 A small airline has just purchased a computer for its new automated reservations system. You have been asked to program the new system. You are to write a program to assign seats on each flight of the airline's only plane (capacity: 10 seats).
